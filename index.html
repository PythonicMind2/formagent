Three steps, zero cost

01
Paste the form URL
Drop in any public Google Form link. No login, no API key, no setup at all.
02
Free AI reads & answers everything
FormAgent fetches the form, extracts every question, and sends them to Pollinations.ai free AI which generates accurate, personalised answers.
03
Auto-fill or follow the guide
With the extension installed — click Auto-Fill Form and watch AI fill the real form in a new tab automatically. Without the extension — follow the colour-coded answer guide manually.
04
Install the Chrome Extension (one time only)
Download the extension zip below. Then:
1. Go to chrome://extensions
2. Enable Developer Mode (top right)
3. Click Load unpacked → select the extension folder
4. The FormAgent icon appears in your toolbar ✅
