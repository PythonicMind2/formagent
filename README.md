# FormAgent - AI Form Filler

Auto-fill any Google Form using free AI. No API key needed.

## 🌐 Live Website
Hosted on GitHub Pages — see the URL in the repo settings after enabling Pages.

## ✨ Features
- 100% free — no API key, no account
- Fills text, multiple choice, checkboxes, dropdowns, date & time
- Smart AI matching with fuzzy + acronym fallback
- Multi-section forms supported
- Chrome extension for one-click auto-fill

## 🚀 How to use
1. Open the website
2. Paste a Google Form URL
3. Click **Generate AI Answers**
4. Click **Auto-Fill Form in New Tab**

## 🔧 Extension Install
1. Download the `extension/` folder
2. Go to `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked** → select the `extension` folder

## 🛠 Tech Stack
- Vanilla HTML/CSS/JS (single file, no framework)
- Pollinations.ai (free AI, no key needed)
- Chrome Extension (Manifest V3)
